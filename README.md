# The Weekly Output

### Development Environment

Initial Setup:

```bash

> git clone git@github.com:MeksPwnschlager/TWO.git TWO
> cd TWO/
> git submodule init
> git submodule update
> make dev

```

Running:

```bash

> make run-dev

```

Misc. Commands:

```
# git
make commit
make push
make pull

# clean venv and dependencies
make clean

# install dependencies
make dev

# migrate django
make migrate
```
